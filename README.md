# SOCIAL MEDIA MARKET PLACE FULL STACK APP

## To Run locally:

Open a terminal in the backend folder and run `npm run dev`

Open a terminal in the frontend folder and run `npm run dev`

Then you can open your browser with the URL `http://localhost:5173/`

If you want to just interact with the backend then you can use an app like `Postman` and then use diffent URLs with different methods such as GET and POST. For example, the posts endpoints can be used with the URL: `http://localhost:5000/api/posts`.
